** How to Install?
#+BEGIN_SRC sh
mv ~/.emacs.d ~/.emacs.d.bak
git clone --depth 1 https://github.com/Wjinlei/.emacs.d.git ~/.emacs.d
# 克隆下来后,这里直接断网启动emacs,会自动安装离线插件包
#+END_SRC
or download the zip package directly and extract to ~/.emacs.d.
